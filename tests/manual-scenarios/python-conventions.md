# python-conventions — manual scenarios

These prompts should activate `python-conventions`. A Python TDD task should also activate `running-tdd-cycles`. A Python review should also activate `reviewing-changes`.

1. "Set up tooling for this Python project."
2. "Configure ruff and mypy strict for this repo."
3. "Is this Python code idiomatic for our stack?"
4. "Add a test.py and run a one-off API check — don't use heredocs."
5. "What linter rules should I enable for a new Python service?"
